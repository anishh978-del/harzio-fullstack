
## About

This project is an Employee Login and Registration Portal built using Firebase Authentication.

It allows employees to register, login using email and password, login with Google, and access a protected security dashboard after authentication.

## Features

- Employee registration
- Employee login
- Google login authentication
- Firebase Authentication integration
- JWT/token-based user session
- Protected dashboard page
- Logout functionality
- Security dashboard UI
- Recent login activity section
- Employee access details
