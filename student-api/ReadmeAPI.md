# Employee Management System

A full-stack Employee Management System built during internship practice.  
The project allows users to register, login, and manage employee records with CRUD operations.

## 🚀 Features

- User Registration
- User Login Authentication
- Secure API access
- Add Employee
- View Employee List
- Update Employee Details
- Delete Employee
- MongoDB Database Integration
- Responsive Frontend UI

## 🛠️ Technologies Used

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Node.js
- Express.js
- REST API

### Database
- MongoDB
- Mongoose

### Authentication
- JWT Authentication
- Password Encryption using bcrypt
